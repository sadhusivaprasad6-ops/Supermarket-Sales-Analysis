# 🛒 Supermarket Sales Analysis

## 📌 Project Overview

This project analyzes supermarket sales data using Python and Jupyter Notebook. It covers data cleaning, exploratory data analysis (EDA), and data visualization to identify sales trends, branch performance, product line performance, customer purchasing behavior, and payment preferences.

The project uses a dataset containing 250 supermarket transactions with sales, profit, tax, product line, customer type, and payment details.

---

## 🎯 Project Objectives

- Load and explore supermarket sales data
- Clean and preprocess the dataset
- Check missing values and duplicate records
- Analyze total sales and profit
- Identify the highest sales branch
- Analyze monthly revenue trends
- Find the best-performing product line
- Analyze customer types
- Analyze payment methods
- Create visualizations for business insights
